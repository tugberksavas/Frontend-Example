Flex Alignments Nedir

Flex alignments, flex container ve flex item'ların hizalanmasını kontrol eden CSS özellikleridir. Flexbox, öğelerin esnek bir şekilde düzenlenmesini sağlar ve bu hizalama özellikleri, öğelerin yatay ve dikey olarak nasıl yerleştirileceğini belirler.

Flex-direction row ise align-items justifty content ne olur?

Flex-direction: row olduğunda, align-items özelliği dikey hizalamayı kontrol ederken, justify-content özelliği yatay hizalamayı kontrol eder. Örneğin:

  • align-items: center; - Flex item'lar dikey olarak ortalanır.
  • justify-content: space-between; - Flex item'lar yatay olarak aralarında boşluk olacak şekilde dağıtılır.
  • justify-content: center; - Flex item'lar yatay olarak ortalanır.
  • justify-content: flex-start; - Flex item'lar yatay olarak başlangıç noktasına hizalanır.
  • justify-content: flex-end; - Flex item'lar yatay olarak bitiş noktasına hizalanır.
  • justify-content: space-around; - Flex item'lar yatay olarak eşit boşluklarla dağıtılır.
  • justify-content: space-evenly; - Flex item'lar yatay olarak eşit boşluklarla dağıtılır, ancak kenar boşlukları da eşittir.
