CSS Kullanim Yerleri

Internal CSS

İç CSS, bir HTML dosyasının içine, etiketi kullanılarak yazılan CSS stil kodlarıdır. Bu stil tanımları genellikle HTML belgesinin head (başlık) kısmında yer alır.

Inline CSS

Inline CSS, bir HTML elementinin style attribute'ini kullanarak CSS kodlarını yazmaktır. Bu stil tanımları genellikle HTML elementinin style attribute'inde yer alır.

Inline CSS

External CSS

Dış CSS, bir HTML dosyasının içine, etiketi kullanılarak yazilan CSS stil kodlarıdır. Bu stil tanımları genellikle HTML belgesinin head (başlık) kısmında yer alır.
